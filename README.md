# SAE14 — Portfolio web

Projet réalisé dans le cadre de la **SAE14** du BUT **Réseaux & Télécommunications**.

## Objectif du projet
L’objectif de cette SAE est de concevoir un **site web portfolio statique** permettant de présenter :
- le profil de l’étudiant,
- ses compétences,
- son parcours,
- ses projets réalisés en BUT R&T.

Le site doit être clair, structuré, lisible et conforme aux bases du développement web.

---

##  Technologies utilisées
- **HTML** : structuration sémantique du contenu
- **CSS** : mise en forme, charte graphique claire (blanc cassé / beige), responsive design
- **JavaScript** : interactions simples (année dynamique, gestion du contact)

Aucune bibliothèque ou framework externe n’a été utilisé.

##  Fonctionnalités principales
- Présentation du profil et des compétences
- Section projets (SAE14, SAE15)
- Navigation simple entre les sections
- Design clair et épuré
- Adaptation aux différents supports (PC, tablette, mobile)
- Contact via lien mailto

##  Compétences mobilisées
- Bases du développement web (HTML / CSS / JavaScript)
- Organisation d’un site web statique

- Structuration et hiérarchisation de l’information
